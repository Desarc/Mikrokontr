## Build instructions
# Drivers
Edit the destination IP in the compile_and_upload_drivers.sh script in the driver dir.
Run the script.
# Game
Again edit the destination IP in the main Makefile in the sokoban dir.
Run make to compile.
Run make send to upload to the board.

## Run instructions
# Drivers
Load the drivers by running drivers_load.sh.
# Game
Start the game by running sokoban.
